Web Development Backend using PHP & MYSQL

About Course

In this course, you will learn how to write codes that handle how the backend and functionality of your application works. You will learn how to send and retrieve forms and other information from databases. You would be taught how to use MySQL Database, as well as communicating with your MySQL Database using the CRUD (Create, Read, Update, Delete) approach.


Who this course is for:

  1. Anyone who wants to start a career as a web developer or backend programmer
  2. Anyone who wants to learn how to build robust web-based applications
  3. Anyone who wants to understand how the various functionalities ofa web-based application interact with one another
  4. Anyone who aspires to become a full-stack developer

You will learn how to:

Create and connect to database

Build a database driven website, with registration, login, authentications and security sessions.

Build an e-commerce website

Integrate with external APIs and payment systems

Build a career portal and customised content management website
