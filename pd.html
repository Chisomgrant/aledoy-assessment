Product Design (UI/UX)

About Course

Learn the concept of Design Thinking, User Interface Design, User Experience Creation, Wireframes & Prototypes, Mockups, Figma.

The User Interface (UI) is the point of human-computer interaction and communication on a device, webpage, or application. User interfaces enable users to effectively control the computer, device or application they are interacting with. UI Designers are generally responsible for collecting, researching, investigating and evaluating user requirements. As a UI designer, it is your responsibility to deliver an outstanding user experience and provide an exceptional and intuitive application design. UI designers create the look and feel of an application's interface including screen layout, animations, transitions, and every single bit of micro-interaction.

On the other hand, the User Experience (UX) is a function of how easy - or difficult - it is to interact with the elements that have been designed on the interface. User Interface and User Experience designers are in high demand and are viable positions across industries and sectors. With more attention being paid to customer experience in marketing, organizations are now more deliberate about enhancing the interface they dish out to customers. By Ideating the branding, design, utilization and function of the product, UX designers set the tone of the overall user experience.

You will learn how to:

Create a style guide for your fonts

Select your colors and create a guide for use of colors

Gather requirements and create user flow for your design

Create a pleasurable experience for your users

Design an app for mobile - Android

Design an app for mobile - iOS

Design an app for web - desktop

Create prototypes for your design
