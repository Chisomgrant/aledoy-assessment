Wordpress & Graphics Design

About Course

WordPress is web publishing software you can use to create a beautiful website or blog. It just may be the easiest and most flexible blogging and website content management system (CMS) for beginners. With WordPress, you can create and run a website without coding. WordPress can be used to create dynamic and mobile-responsive websites with numerous features. The websites of some renowned brands like TechCrunch, Playstation, Skype, Walt Disney, Flickr, CNN and a host of others were built with Wordpress. Statistics has it that 34% of the web uses WordPress WordPress gives room for the installation of various plugins to help achieve your desired outcome. During the course, you would also be taught how to build e-commerce websites similar to the likes of Amazon, Jumia, and Konga. In this course, students will learn how to build websites for their businesses and for clients and start making money immediately.


Requirements:

  1. No prior knowledge of WordPress is required as everything will be covered in this course.
  2. You will need a working computer and a web browser connected to the internet.
  3. You don't need any coding experience at all.

You will learn how to:

Install Wordpress on PC

Setup and customise your WordPress website

Create pages and add contents to your pages

Setup a blog using WordPress

Add Security & Privacy settings to your WordPress website

Back up your WordPress website

Make your website SEO friendly

Create E-Commerce websites using Opencart

Register and host your website
